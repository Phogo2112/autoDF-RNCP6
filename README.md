# AutoDF-Front
# autoDF Monorepo

Ce dépôt contient le projet `autoDF` sous forme de **monorepo** avec le backend (`API`) et le frontend (`Front`).  
Le projet est conçu pour être lancé facilement via **Docker** et servi par **Nginx**.

---

## Prérequis

- [Docker](https://www.docker.com/get-started) installé
- [Docker Compose](https://docs.docker.com/compose/install/) installé
- Git (pour cloner le projet)
